Noise Plugin Installation instructions:
	Drag all the .class files into your ImageJ program directory’s plugins folder (ImageJ > plugins)
	Drag the Noise.ijm file into your ImageJ program directory’s macros folder (ImageJ > macros)
	If ImageJ is already open on your computer, do one of the following two options:
		Restart ImageJ (preferred)
		Go to Help > Refresh Menus

How to use:
	Go to Plugins > Noise (should be at the bottom)
	Select the folder of images you wish to analyze
	Instructions then follow on the screen
